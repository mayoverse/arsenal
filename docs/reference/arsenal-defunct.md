## Details

`comparison.control` was renamed to
[`comparedf.control`](https://mayoverse.github.io/arsenal/reference/comparedf.control.md)
in version 3.0.0.

`compare.data.frame` was renamed to
[`comparedf`](https://mayoverse.github.io/arsenal/reference/comparedf.md)
in version 3.0.0.

`length.tableby` was removed in version 2.0.0.

`includeNA.character` and `includeNA.numeric` were removed in version
2.0.0 and replaced with a default method.

`rangeTime` was removed in version 1.5.0.
